# Python-avanzado
Tarea programación python avanzado

Esta práctica del módulo “Programación Python (avanzado)” tiene como objetivo afianzar los conocimientos vistos en “Fundamentos de Python” y “Python Avanzado” mediante la generación de un dataset a partir de un algoritmo que crea hostnames aleatorios siguiendo reglas concretas, su transformación a un DataFrame de Pandas y la creación de gráficos con Matplotlib; la entrega puede ser parcial y se considera aprobada con una nota de 5 o más puntos, apoyándote en los ejemplos y documentación del módulo. Los requisitos se puntúan por apartados: primero debes importar las librerías necesarias (+0,15) e inicializar variables que luego modificarás (+0,15); después crear una función set_hostnames(number_of_hosts:int) (+1,5) que genere una lista de hostnames de 8 caracteres alfanuméricos en mayúsculas, donde el primer carácter indica el sistema operativo con proporciones aproximadas Linux (L) 40%, Solaris (S) 30%, AIX (A) 20% y HP-UX (H) 10%, el segundo carácter indica el entorno con proporciones Development (D) 10%, Integration (I) 10%, Testing (T) 25%, Staging (S) 25% y Production (P) 30%, los tres siguientes caracteres indican el país (NOR, FRA, ITA, ESP, DEU, IRL) con proporciones Norway 6%, France 9%, Italy 16%, Spain 16%, Germany 23% e Ireland 30%, y los tres últimos dígitos representan el número de nodo incremental para cada combinación de sistema operativo–entorno–país, comenzando en 001 y con máximo 999. A continuación debes implementar funciones de parseo: get_os(hostname:str) (+0,5) que devuelva “Linux”, “Solaris”, “AIX” o “HP-UX” según la primera letra, o “Unknow” si no encaja; get_enviroment(hostname:str) (+0,5) que devuelva “Development”, “Integration”, “Testing”, “Staging” o “Production” según la segunda letra, o “Unknow” si no coincide; y get_country(hostname:str) (+0,5) que devuelva “Norway”, “Germany”, “Italy”, “Spain”, “Ireland” o “France” según los caracteres 3 a 5 del hostname, o “Unknow” si no corresponden a los códigos indicados. Luego debes crear la función set_dataframe(count:int) (+1) que, usando una variable global df inicializada como None, llame a set_hostnames(count), recorra la lista generada y vaya añadiendo a una lista global dataset (inicializada como lista vacía) un diccionario por cada hostname con las claves hostname (ej. “LDIRL003”), os (ej. “Linux”), enviroment (ej. “Development”), country (ej. “Ireland”) y node (entero, ej. 3), para finalmente construir un DataFrame de Pandas con esa lista de diccionarios y asignarlo a la variable global df. En el paso siguiente debes crear el DataFrame invocando set_dataframe(1500) (+0,2) e inspeccionar df para comprobar que se ha generado correctamente. Después debes guardar el DataFrame en un CSV llamado hosts.csv en la misma carpeta del notebook, con header=True e index=False (+0,5), y probar a leerlo con read_csv guardándolo en una variable hosts_df y visualizándolo. En la parte de visualización, debes generar un gráfico único (+0,5) agrupando por country y enviroment, usando unstack y un gráfico de barras (kind=bar). Finalmente, el bloque principal (+4,5) consiste en crear una figura con 4 gráficos en una cuadrícula 2x2: arriba a la izquierda, un gráfico de barras horizontales titulado “Type of OS grouped by country” que agrupe por país y sistema operativo usando groupby, unstack y barh; arriba a la derecha, un gráfico de tarta titulado “Total Operating Systems” con el total de sistemas operativos del DataFrame, usando groupby y pie, mostrando como labels el número de sistemas por tipo y añadiendo una leyenda en la esquina superior derecha con el porcentaje por sistema; abajo a la izquierda, un gráfico de barras horizontales titulado “Total hosts by country” calculado con value_counts() de country, con etiquetas de ejes “Number of hosts” (x) y “Country” (y), mostrando además el número total de hosts a la derecha de cada barra, y ajustando el máximo del eje x al valor de la barra mayor +100 para dejar margen (opcionalmente aplicando paleta con seaborn); y abajo a la derecha, un gráfico titulado “Hosts by country grouped by enviroment” que agrupe hosts por país y entorno usando groupby, unstack(0) y bar, añadiendo como etiqueta del eje y “Number of hosts”, terminando con el ajuste de espacios con fig.tight_layout().
